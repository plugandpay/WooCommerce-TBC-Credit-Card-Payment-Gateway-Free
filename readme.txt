== Description ==

Accept Visa/Mastercard payments in your WooCommerce shop using TBC gateway.

= Free =
* SMS transaction allows you to capture funds instantly
* Close business day
* Autogenerated OK/FAIL urls
* Detailed logs

= Premium =
* DMS transaction allows you to block required amount first and capture funds later
* Refunds
* Multi merchant support allows you to accept GEL, USD or EUR
* Support (mail/phone)

Learn more about our [premium plugin](https://plugandpay.ge)
